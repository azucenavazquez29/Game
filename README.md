# Hangman English Game 🎮

This is a simple Hangman game to practice English grammar and vocabulary.

## ✨ Features
- Zero conditional phrases
- Present perfect sentences
- Body parts vocabulary
- 4 attempts, first letter revealed

## 🚀 How to play
1. Open the game in your browser.
2. Guess letters by typing them.
3. If you complete the phrase before running out of attempts, you win!
